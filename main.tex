\subsection{詰將棋（局部）盤面}
使用指令
\begin{lstlisting}
\tsumeshogi{#1}{#2}{#3}{#4}{
#5
}
\end{lstlisting}
分別指定限制的直行（筋）從第 \#1 筋到第 \#2筋為止；\\
限制的橫列（段）從第 \#3 段到第 \#4段為止。\\
\#5 放置所有使用到的棋駒。
\begin{lstlisting}
\tsumeshogi{1}{4}{1}{5}{
    \koma11\Ky \koma13\Fu \koma22\Gy
    \koma23\KA \koma42\Fu
    \mochigoma\HI\mochigoma[2]\KI
}
\end{lstlisting}
